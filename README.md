# Job Portal

## Overview

Job Portal is a simple web-based application built using HTML, CSS, and JavaScript. The project provides a clean and user-friendly interface where users can select their role and log in to the system.

The goal of this project is to demonstrate a basic job portal interface with role-based access such as Applicant, Recruiter, and Admin. It focuses mainly on frontend design and user interaction.

---

## Features

* Role-based login selection
* Three user roles: Applicant, Recruiter, and Admin
* Clean and modern user interface
* Gradient background design
* Simple login system using JavaScript
* Responsive layout for better viewing

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
